# Components using NextJS and TainWindCSS

## Getting Started

```sh
# install packages
yarn

# start storybook
yarn sb

# start nextjs
yarn dev
```

## Tailwind CSS

- [daisyUI](https://daisyui.com/)

- [Flexbox & Grid](https://tailwindcss.com/docs/flex-basis)
